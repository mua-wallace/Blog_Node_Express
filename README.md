# Blog_Node_Express
This Blog_Node_Express is site where users can  create a blog, see all blogs, see content of blog and delete blog etc
######
# Installation
- ``` git clone https://github.com/mua-wallace/Blog_Node_Express ```
- ```cd Blog_Node_Express ```
- ``` npm install ```
- ``` npm run dev ```
